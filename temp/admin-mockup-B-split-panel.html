🟢 Option B — Key Changes vs. Current Design

  • [new] Persistent left panel for subject navigation (folder tree + map count per folder)
  • [changed] Maps shown as horizontal list rows instead of grid cards — easier to scan titles and compare status
  • [new] Per-subject header bar with map count breakdown (built-in / local) and subject-scoped actions
  • [new] Filter chips: All / Built-in / Custom / Draft — filter within current subject
  • [changed] Drag handles on row left side — clear and always visible, no text explanation needed
  • [changed] Actions per row are contextual: built-in shows just "Open"; custom shows Unpublish + Export + Open
  • [removed] All danger/utility actions (Reset Storage, Export Manifest) moved out of main view
  • [new] Subject search input for quickly finding a subject when the list grows long
