🟢 Student sidebar — what stays

  • Same "Topics" label + Collapse/Expand controls
  • Overall progress chip below the header
  • Subject folders (collapsible) with map items + per-map progress bar
  • Admin entry at the bottom (only visible when unlocked)
  • Clicking Admin entry → switches to Admin mode

🟡 Admin sidebar — what changes

  • Label becomes "Subjects" instead of "Topics"
  • Progress chip replaced by subject search input
  • Items become subject nav entries (icon + count)
  • "All Maps" entry at top for unfiltered view
  • Footer gains "+ New Subject Folder" button

🔵 What's shared / stays identical

  • Sidebar width (260px), background color, border
  • Topbar layout — only the role-btn highlight color changes (teal → amber)
  • Collapse/expand behavior (sidebar can still be hidden on small screens)
  • The app-shell grid template doesn't change at all
  • Role switcher in topbar is the toggle — no separate navigation needed
  • Switching back to Student re-shows the last-active map seamlessly
