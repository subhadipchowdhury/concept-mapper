  • [changed] Larger, taller cards with gradient color bands instead of a thin top stripe — much more distinct and visually rich
  • [new] Subject tag shown inside the card band (breadcrumb-style) so subject context is always visible in "All" view
  • [removed] Folder section accordion structure replaced by filter state — less nested, flatter browsing experience
  • [changed] Status badge moved to top-right corner of card title area for immediate visibility
  • [changed] Actions (Publish, Export) only visible for custom maps; built-in cards are intentionally minimal
  • [changed] Overflow menu (⋯) for secondary/danger actions — same approach as Option A
